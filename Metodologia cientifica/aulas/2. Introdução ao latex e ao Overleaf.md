- Formato padrão IEE (de bibliotecas):
```Latex
\documentclass[conference]{IEEEtran}
\IEEEoverridecommandlockouts
% The preceding line is only needed to identify funding in the first footnote. If that is unneeded, please comment it out.
\usepackage{cite}
\usepackage{amsmath,amssymb,amsfonts}
\usepackage{algorithmic}
\usepackage{graphicx}
\usepackage{textcomp} %duas colunas
\usepackage{xcolor}
\def\BibTeX{{\rm B\kern-.05em{\sc i\kern-.025em b}\kern-.08em
    T\kern-.1667em\lower.7ex\hbox{E}\kern-.125emX}}
\begin{document}
```

- Pode mudar para o formato de conference para journal sem mudar o template: 
	- Antes (conference):
	```Latex
	\documentclass[conference]{IEEEtran}
	```
	- Depois(Journal):
	```Latex
	\documentclass[journal]{IEEEtran}
	```

### Seções
- Criar seção:
```Latex
\section{Titulo_seção}
```
- Subseção:
```Latex
\subsubsection{Trabalhos relacionados}
% cada subsub é indicando que é uma subseção da subseção
```

### Acrônimos
- Todo artigo deve ter um acrônimo, como adiciona-los: ![[Captura de Tela 2026-03-17 às 16.04.25.png]]
- Criar um arquivo ``acronimos.txt``:
	```Latex
	\acrodef{6G}{Sexta Geração de Redes Móveis}
	%acrodef {acronimo}{Palavra completa}
	```
- Na main: (antes da introdução)
```Latex
\input{acronimos}
```
-  Usar acrônimo:
```Latex
A tecnologia \ac{6G} revolucionará o mercado.
```
	- Criar o arquivo
	- chamar o arquivo 
	- chamar \ac

### Referencias
- Como pegar elas no google academico: (clica em citar), depois em bibtex e copia tudo que aparece lá
![[Captura de Tela 2026-03-17 às 16.21.40.png]]
- É necessário criar um arquivo "referencias.bib" (onde será colado o que foi copiado em BibTeX)
	- Formato do arquivo .bib:
	```Latex
	@article{ahmad2019security,
  title={Security for 5G and beyond},
  author={Ahmad, Ijaz and Shahabuddin, Shahriar and Kumar, Tanesh and Okwuibe, Jude and Gurtov, Andrei and Ylianttila, Mika},
  journal={IEEE Communications Surveys \& Tutorials},
  volume={21},
  number={4},
  pages={3682--3722},
  year={2019},
  publisher={IEEE}
}
	```
- no main pra usar as referências:
	```Latex
	\cite{ahmad2019security}
	\bibliographystyle{IEEEtran}
	\bibliography{referencia}
```
