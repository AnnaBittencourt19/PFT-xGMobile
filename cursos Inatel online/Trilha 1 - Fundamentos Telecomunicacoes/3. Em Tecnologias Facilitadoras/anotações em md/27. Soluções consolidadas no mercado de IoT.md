### Fundamentos em telecomunicações em tecnologias facilitadoras
- Fundamentos em IoT
- Tema principal dessa apresentação: soluções consolidadas no mercado de IoT, com foco em plataformas de gerenciamento, Digital Twin, agricultura de precisão, IoT na saúde e gateways IoT
### Introdução
- À medida que a Internet das Coisas se expande, surgem soluções consolidadas para atender diferentes setores da indústria e dos negócios
- IoT significa Internet of Things
- Em português, Internet das Coisas
- A ideia central do IoT é conectar dispositivos físicos, sensores, máquinas, sistemas e plataformas para coletar dados e transformar esses dados em ações úteis
- A apresentação mostra que a IoT já não é apenas uma tendência futura
- Ela já está presente em empresas, indústrias, agricultura, saúde, energia, logística e telecomunicações
- Essas soluções se destacam porque ajudam a:
	- Integrar dados
	- Otimizar operações
	- Reduzir custos
	- Melhorar produtividade
	- Criar novos serviços
	- Aumentar eficiência operacional
	- Promover inovação
### Por que estudar soluções consolidadas de IoT?
- Entender soluções consolidadas é importante porque mostra como a IoT sai da teoria e aparece na prática
- Em vez de olhar apenas para sensores e redes, o foco passa a ser como empresas usam IoT para resolver problemas reais
- As soluções de IoT normalmente envolvem:
	- Dispositivos conectados
	- Sensores
	- Gateways
	- Plataformas em nuvem
	- Análise de dados
	- Inteligência artificial
	- Automação
	- Monitoramento remoto
	- Integração com sistemas empresariais
- Com a IoT se tornando parte da estratégia de negócios, conhecer essas soluções ajuda qualquer organização a se manter competitiva no cenário digital
### IoT como estratégia de negócios
- A IoT não serve apenas para conectar objetos
- Ela também cria valor para empresas e usuários
- Uma empresa pode usar IoT para entender melhor seus processos, reduzir falhas, prever problemas e tomar decisões com base em dados reais
- Em ambientes industriais, por exemplo, sensores podem indicar falhas antes que uma máquina pare
- Na agricultura, sensores podem indicar quando irrigar ou ajustar o manejo
- Na saúde, dispositivos conectados podem monitorar temperatura, pacientes e equipamentos médicos
- Por isso, IoT é uma tecnologia que conecta o mundo físico ao digital e melhora a tomada de decisão
### Plataformas de gerenciamento de dispositivos IoT
- Plataformas de gerenciamento de dispositivos IoT são sistemas usados para conectar, monitorar, controlar e analisar dispositivos conectados
- Elas funcionam como uma base central para organizar o ecossistema IoT
- Em uma solução IoT real, não basta instalar sensores
- É necessário gerenciar esses sensores, receber dados, processar informações e integrar tudo com aplicações
- As plataformas de IoT ajudam nessa parte
- Elas permitem:
	- Conectar dispositivos à nuvem
	- Coletar dados
	- Processar informações em tempo real
	- Fazer análise de dados
	- Monitorar dispositivos
	- Enviar comandos
	- Integrar com outros serviços
	- Escalar para muitos dispositivos
### AWS IoT Core
- AWS IoT Core é a plataforma da Amazon Web Services voltada para conexão de dispositivos IoT à nuvem
- Ela permite conectar dispositivos, coletar dados e integrar esses dados com outros serviços da AWS
- A plataforma oferece suporte para bilhões de dispositivos
- Também possui ferramentas para processamento e análise de dados em tempo real
- Isso é importante porque aplicações IoT podem gerar grande volume de dados
- Exemplos de uso:
	- Monitoramento industrial
	- Dispositivos conectados
	- Automação
	- Logística
	- Veículos conectados
	- Coleta de dados em larga escala
### Microsoft Azure IoT Hub
- Microsoft Azure IoT Hub é um serviço de gerenciamento de IoT da Microsoft
- Ele facilita a comunicação bidirecional entre dispositivos IoT e a nuvem
- Comunicação bidirecional significa que o dispositivo pode enviar dados para a nuvem e também receber comandos da nuvem
- A plataforma se integra a outras soluções da Azure
- Exemplos:
	- Azure AI
	- Azure Machine Learning
- Essa integração é importante porque permite usar inteligência artificial e aprendizado de máquina sobre os dados coletados pelos dispositivos
### Google Cloud IoT Core
- Google Cloud IoT Core é apresentado como uma plataforma para gerenciar dispositivos IoT
- Sua função é permitir coleta, processamento e análise de dados em grande escala
- Em soluções IoT, grande escala é um ponto essencial porque uma empresa pode ter milhares ou milhões de dispositivos conectados
- Uma plataforma em nuvem ajuda a centralizar os dados e transformar informações brutas em análises úteis
- Isso facilita o monitoramento, a automação e a tomada de decisão

### Por que plataformas em nuvem são importantes para IoT?
- A nuvem fornece processamento, armazenamento e integração para soluções IoT
- Muitos dispositivos IoT são pequenos e possuem pouca capacidade computacional
- Por isso, eles coletam dados e enviam para a nuvem
- A nuvem pode:
	- Armazenar grandes volumes de dados
	- Executar análises
	- Rodar modelos de inteligência artificial
	- Gerar alertas
	- Criar dashboards
	- Integrar sistemas
	- Permitir acesso remoto
- Sem uma plataforma de gerenciamento, uma solução IoT grande fica difícil de operar

### Digital twin
- Digital twin significa gêmeo digital
- Um gêmeo digital é uma réplica virtual de um ambiente, processo, sistema ou equipamento físico
- Essa tecnologia é considerada um marco da Indústria 4.0
- Ela usa IoT para coletar dados do mundo real e projetar esses dados em um ambiente digital
- Sensores e câmeras capturam informações do objeto físico
- Depois, essas informações alimentam uma versão digital que pode ser analisada e simulada
### Como funciona um Digital Twin?
- Um Digital Twin depende de dados reais
- O processo básico é:
	- Sensores coletam dados do equipamento ou ambiente físico
	- Esses dados são enviados para uma plataforma digital
	- A plataforma cria ou atualiza a réplica virtual
	- A réplica permite monitorar, simular e prever comportamentos
	- Os resultados ajudam na tomada de decisão
- O gêmeo digital não é apenas um desenho 3D
- Ele precisa estar conectado aos dados reais do sistema físico
### Para que serve um Digital Twin?
- Um gêmeo digital permite testar cenários antes de aplicar mudanças no mundo real
- Isso reduz riscos e custos
- Exemplos de uso:
	- Simular operação de uma fábrica
	- Prever falhas em equipamentos
	- Testar mudanças em uma refinaria
	- Monitorar máquinas em tempo real
	- Avaliar desempenho de processos
	- Otimizar produção
	- Treinar equipes em ambiente virtual
### Digital Twin e Indústria 4.0
- A Indústria 4.0 usa tecnologias digitais para tornar processos industriais mais inteligentes
- O Digital Twin se encaixa nesse contexto porque conecta o físico ao digital
- Ele permite que empresas acompanhem o comportamento de equipamentos e ambientes com mais precisão
- Também possibilita decisões mais rápidas e baseadas em dados
- Em vez de esperar uma falha acontecer, a empresa pode simular cenários e agir antes
### Exemplo: Petrobras e Digital Twins
- A apresentação mostra a Petrobras como exemplo de uso de Digital Twins
- A Petrobras utiliza gêmeos digitais para criar réplicas digitais de refinarias
- Com isso, consegue simular diferentes cenários antes da implementação prática
- Esse processo já trouxe mais de 200 milhões de dólares em economia anual
- O objetivo é aumentar a eficiência e reduzir custos operacionais
### Smart Twin
- A Petrobras está avançando para o conceito de Smart Twin
- Smart Twin é uma evolução do Digital Twin com uso de inteligência artificial
- A ideia é que as simulações digitais otimizem resultados de forma mais rápida
- Com IA, o sistema pode ajudar a tomar decisões autônomas
- Isso significa que o gêmeo digital deixa de ser apenas um ambiente de simulação e passa a apoiar decisões inteligentes
### Digital Twin x Smart Twin
| Conceito | Ideia principal |
|---|---|
| Digital Twin | Réplica digital de um sistema físico alimentada por dados reais |
| Smart Twin | Gêmeo digital com inteligência artificial para otimização e decisões mais autônomas |
- O Digital Twin permite observar e simular
- O Smart Twin vai além, pois usa IA para sugerir ou tomar decisões de otimização

### IoT na saúde
- A IoT na saúde envolve o uso de dispositivos conectados para monitorar, controlar e melhorar serviços médicos
- Pode incluir sensores, equipamentos hospitalares, wearables, dispositivos de armazenamento e sistemas de acompanhamento remoto
- A saúde é uma área muito sensível porque envolve dados pessoais e condições críticas
- Por isso, soluções IoT na saúde precisam de segurança, confiabilidade e monitoramento constante
### Vaccine Smart Fridge
- A apresentação usa o Vaccine Smart Fridge como exemplo de IoT na saúde
- Ele é um refrigerador inteligente projetado para armazenar vacinas de forma eficiente e segura
- Seu objetivo é garantir condições ideais de temperatura e monitoramento constante
- Isso é essencial porque vacinas precisam ser armazenadas em uma faixa adequada de temperatura
- Se essa condição for perdida, a eficácia da vacina pode ser comprometida
### Cadeia de frio de vacinas
- Cadeia de frio é o conjunto de processos usados para manter vacinas em temperaturas adequadas desde a fabricação até a aplicação
- A IoT ajuda nesse processo porque permite monitorar temperatura em tempo real
- Se a temperatura sair da faixa ideal, o sistema pode gerar alerta
- Isso evita perdas e aumenta a segurança do armazenamento
- Esse tipo de solução é ainda mais importante em locais com energia instável ou onde o controle manual é difícil
### Benefícios da IoT na saúde
- Monitoramento constante
- Maior segurança para medicamentos e vacinas
- Redução de perdas
- Alertas automáticos
- Rastreamento de condições críticas
- Apoio ao cuidado remoto
- Melhor controle de equipamentos
- Mais confiabilidade em processos hospitalares
### Gateways IoT
- Gateways IoT são equipamentos intermediários entre dispositivos conectados e plataformas de processamento
- Eles recebem dados de sensores e dispositivos
- Depois encaminham esses dados para a nuvem, servidor local ou sistema de controle
- Também podem fazer processamento local, tradução de protocolos e gerenciamento de dispositivos
- Em ambientes industriais, gateways são muito importantes porque ajudam a conectar dispositivos diferentes e operar em condições difíceis
### Funções de um gateway IoT
- Um gateway IoT pode:
	- Coletar dados de sensores
	- Traduzir protocolos
	- Encaminhar dados para a nuvem
	- Fazer processamento local
	- Gerenciar dispositivos
	- Aplicar regras de segurança
	- Reduzir tráfego enviado para a nuvem
	- Permitir edge computing
### Edge computing em gateways
- Edge computing significa processar dados perto da origem
- Em vez de enviar tudo para a nuvem, o gateway pode processar parte dos dados localmente
- Isso reduz latência e tráfego de rede
- Em ambientes industriais, edge computing é importante porque algumas decisões precisam ser rápidas
- Exemplo:
	- Um sensor detecta uma falha crítica
	- O gateway processa o alerta localmente
	- O sistema age rapidamente sem esperar uma nuvem distante
