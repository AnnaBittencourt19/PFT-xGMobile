Para verificar se uma string contém um caractere, use o operador de associação `in` — por exemplo, `"a" in word` retorna `True` se `"a"` for encontrado dentro de `word`.  
  
Isso **não** é o mesmo que a palavra-chave `in` usada em um loop `for` — aqui, `in` simplesmente verifica se um valor existe dentro de outro (como verificar se um dígito existe dentro de uma string).  
  
Como o `in` funciona em strings, você deve primeiro converter o número para uma string usando `str(num)` — por exemplo, `"3" in str(num)` verifica se o dígito 3 aparece em qualquer lugar do número.
