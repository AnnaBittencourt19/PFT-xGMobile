### Função lambda
- Def para coisas mais complexas (que não cabe em uma linha)
- Lambda é para coisa mais simples que cabem em uma linha
```Python
quadrado = lambda x: x**2
#nome variavel = lambda variavel_que_vou_usar_na_função : o que a função deve fazer
```
- Uma função lambda não tem nome porém declaramos ela em uma "variável" para chamar ela
### Lambda + Pandas 
- elementos separados por virgula e dentro de \[\]: Linha
- elementos dentro de uma {} é a coluna
```Python
df = pd.DataFrame({"coluna1":[ "linha1_coluna1 ", "linha2_coluna1", "dados ", " outra frase", " MAIS UM TEXTO"]})
```
- Apply(função lambda): aplica uma função lambda em um dataframe
- Exemplo:
```Python
df['texto_limpo'] = df['texto_bruto'].apply(lambda x: x.lower().strip()) #nova coluna com elementos formatados
```
- df \['nome coluna'\] também cria uma coluna
