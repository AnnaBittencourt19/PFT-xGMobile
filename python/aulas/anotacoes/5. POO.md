- Fábrica de robos
- Classe: Planta do projeto, define as estruturas (projeto, "rascunho")
- Objeto: É o projeto construído a partir da planta (resultado da planta)
- Instanciação: É o "apertar"o botão para criar um objeto a partir da planta
- Atributos: São as caracteristicas de cada objeto
- \_init\_ : Configuração inicial
- Métodos: São as ações que a máquina executa (usam atributos do objeto)
- self: O parâmetro self é a forma da classe referir a si mesmo
- Exemplo:
```Python
class RoboBase:
	def __init__(self, nome):
		self.nome = nome
		self.bateria = 100
	def trabalhar(self):
		self.bateria -= 10
		print(self.nome, 'está trabalhando. Bateria:', self.bateria, '%')

meu_robo = RoboBase('Alpha') #instanciacao

meu_robo.trabalhar()
# em Python tem que usar self para o objeto chamar as coisas dele mesmo
```
- Metodo de instância (self): Afeta apenas um objeto especifico
- Metodo de classe (@classmethod): Afeta todos os objetos (cls no lugar de self)
	- Self só um objeto
	- Cls toda a classe
- Metodo estático (@staticmethod): Um extra, não afeta 


- Herança: Pega uma planta (classe) e cria outra com algo a mais, pega os atributos base e cria uma versão com outras opções. Tipo IPhone 16 é uma classe base, o IPhone 16 pro max é uma herança do 16, ele tem atributos em comum porém com adições

- Ajuste fino 
- Substituição: (Overriding) Classe filha modifica uma ação da classe mãe
	- Utiliza super()
- Polimorfismo: Usar um metodo igual em dois objetos diferentes e a mesma tarefa pode ser executada de formas diferentes 

- Classes abstratas: Molde obrigatório, padronização, força todas as classes filhas a implementarem um método especifico 
	- Tem que importar ABC (from abc import ABC)

- Controle de qualidade: Inspetores de qualidade, if dentro de um \_set_ (verificar erros)
- Get: returna algo
- Delete: Apaga lixo
